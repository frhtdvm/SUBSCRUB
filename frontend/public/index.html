CORE FEATURES

🔍
Detection Engine
Recurring pattern detection across bank transactions and email receipts. Confidence scoring with 500+ directory entries.
💸
Waste Calculator
Monthly equivalent cost, annual projection, and "potential leak" identification for suspected but unconfirmed charges.
📋
Legal Templates
GDPR, KVKK, and Generic jurisdiction cancellation letters, generated on-device, ready to send via Mail Composer.

Local-Only Storage
SQLite with SQLCipher encryption. No analytics, no crash reporting, no remote logging. Data never leaves the device.

Demo Mode
Full app flow works without any API keys. 500+ demo transactions and emails seeded locally at first launch.
💰
RevenueCat Paywall
Premium features (legal templates, direct links) gated behind a native paywall powered by RevenueCat.

HOW TO RUN

  1. Clone and install dependencies
    cd /app/SubScrub
npm install
  2. Start Expo (Demo Mode — no credentials needed)
    npx expo start
    Scan QR code with Expo Go on iOS or Android. Full demo flow: Splash → Onboarding → ConnectSources → Scan → Dashboard.
  3. Development build (for SQLCipher + RevenueCat)
    npm install -g eas-cli
eas build --profile development --platform ios
  4. Set production credentials (when ready)
    eas secret:create --scope project \
  --name EXPO_PUBLIC_REVENUECAT_API_KEY --value appl_...
    See .env.example for all required vars.
  5. Submit to App Store / Play Store
    eas build --profile production --platform all
eas submit --profile production --platform all
